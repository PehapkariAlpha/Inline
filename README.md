Requirements
------------

Pehapkari/InlineEditable requires PHP 7.0 or higher


Installation
------------

The best way to install Pehapkari/InlineEditable is using [Composer](http://getcomposer.org/):

```sh
$ composer require pehapkari/inline-editable
```
