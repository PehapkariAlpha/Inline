Requirements
------------

XcoreCMS/InlineEditing requires PHP 7.1 or higher


Installation
------------

The best way to install XcoreCMS/InlineEditing is using [Composer](http://getcomposer.org/):

```sh
    composer require xcore/inline-editing
```
